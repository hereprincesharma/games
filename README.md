# 🎬 Movie Guessing Game

A simple two-player movie guessing game made using **JavaScript**.

## 📌 Project Description

In this game, there are two players:

* **Player 1** enters a movie name using the JavaScript `const` data type.
* **Player 2** tries to guess the movie name.
* If Player 2 guesses the correct movie name, Player 2 wins the match.
* If the guess is wrong, Player 2 loses the match.

This project is created to practice basic JavaScript concepts in a fun way.

## ✨ Features

* Two-player game
* Movie name guessing
* Win and lose result message
* Uses JavaScript `const` data type
* Simple and beginner-friendly logic

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📚 JavaScript Concepts Used

* `const` data type
* Variables
* Conditional statements (`if-else`)
* User input
* String comparison
* DOM Manipulation
* Event Listeners

## 🎮 How to Play

1. Player 1 enters a movie name.
2. Player 2 enters their guessed movie name.
3. Click on the **Check Guess** button.
4. If both movie names are the same, Player 2 wins.
5. If the names are different, Player 2 loses.


## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open the `index.html` file in your browser.
4. Start playing the movie guessing game.
